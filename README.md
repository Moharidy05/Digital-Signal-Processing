This repository contains a simple, menu-driven Python framework for loading, visualizing, and performing basic arithmetic operations on one-dimensional signal data. It leverages **NumPy** for efficient numerical handling and **Matplotlib** for comprehensive plotting.

---

## ğŸš€ Features

* **Data Loading:** Load signal data (index and amplitude pairs) from a plain text file.
* **Visualization:** Display a signal simultaneously in **continuous** (line plot) and **discrete** (stem plot) representations.
* **Comparison:** Plot two signals together to visually compare them.
* **Arithmetic Operations:**
    * **Addition:** Add two or more signals together (supports signals of different lengths via zero-padding).
    * **Multiplication:** Amplify, reduce, or invert a signal by multiplying it by a constant factor.
